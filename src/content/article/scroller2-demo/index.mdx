## 데이터 저널리즘의 미래

기존에는 시각화 하나를 만들 때마다 복잡한 HTML 구조를 일일이 짜야 했습니다. 하지만 `Scroller2`를 사용하면 위와 같이 데이터(객체 배열)만 정의하면 됩니다.

<Scroller2 steps={scrollerSteps}>
  <div slot="vis" id="demo-vis-container" class="w-full h-full border border-dashed border-gray-300 flex items-center justify-center">
    <p class="text-gray-400">여기에 D3 시각화가 렌더링됩니다.</p>
  </div>
</Scroller2>

<script>
  import { initDemo } from "/src/graphics/template/scroller2/js/demo.js";
  initDemo('#demo-vis-container');
</script>

본문 내용이 이어집니다. 이제 콘텐츠 제작자는 코드가 아닌 **'이야기'**에만 집중하면 됩니다.
