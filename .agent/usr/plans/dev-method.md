# 현황

## 코드 품질 분석 (2025-11-27 기준)

- 현재까지 작성된 코드는 비전문가 수준의 디버깅 중심 개발 방법을 채용하고 있다.
- 이로 인한 문제는 다음과 같다.
 - 가장 큰 문제는 앤드루 혹은 다른 에이전트가 제안하는 현업 수준의 코드와 패턴과 구성이 일치하지 않는다.
 - 코드 품질이 균일하지 않다.
 - 향후 코드 개선 시에도 앞서 말한 문제가 코드 개선 속도와 품질을 저하시킬 수 있다.

## 코드 품질 개선 방향

### 단기 과제
<!-- 아래 구성은 애자일을 간략히 살핀 후에 작성한 내용이다. 앤드루는 전통적인 디버깅 중심의 초보자급 코드 작성 습관을 전면 재검토해야 한다. 프로그래밍 현업에 적합한 애자일로 단계적으로 개선할 수 있도록 실천 방안을 제시해야 한다.  -->

- 애자일을 도입한다. 최우선 과제는 다음과 같다.
 - 가장 작지만 가장 중요한 단위부터 선별해 해당 코드 및 연계된 코드를 단계적으로 작성한다.
 - TDD(Test driven development)를 선별적 적용해 효율성을 검증한다.

### 장기 과제

- 애자일을 효과적으로 운용해 기능 개선 속도를 높인다.
- TDD를 효과적으로 운용해 코드 품질을 높인다.
