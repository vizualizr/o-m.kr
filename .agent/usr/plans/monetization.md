# 수익 구조

본 문서는 세부적인 단기 계획을 기재한다.

## 원칙

- 홍보는 수익 창출의 일환이다.


### 홍보

1. **검색 엔진 최적화(SEO)**: 데이터 저널리즘의 특성을 살려 통계, 수치, 시각화 관련 롱테일 키워드를 선점한다. 검색 엔진을 통한 유기적 유입(Organic Traffic)을 극대화하여 광고 수익 기반인 트래픽을 확보한다.

2. **소셜 미디어 마케팅**: 시각적 정보 전달력이 높은 차트와 인포그래픽을 활용해 SNS 상에서 콘텐츠의 확산을 유도한다. 이를 통해 사이트 유입을 늘리고 브랜드 인지도를 제고한다.

3. **뉴스레터 및 리텐션 강화**: 심층 데이터 분석 기사를 뉴스레터로 제공하여 방문자의 재방문을 유도하고, 단순 방문자를 충성 독자로 전환시켜 구독 수익 모델의 기반을 다진다.

4. **퍼스널 브랜딩**: 1인 프로젝트의 강점을 활용해 운영자의 전문성과 저널리즘 철학을 공유하며 독자와의 신뢰 관계를 구축한다.
